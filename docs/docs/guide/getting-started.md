# Getting Started

Welcome to the HTML Email Editor! This tool helps you create professional-looking emails without needing to know HTML code.

## Opening the Editor

1. Open your web browser (Chrome, Firefox, or Edge recommended).
2. Go to [http://localhost:5173](http://localhost:5173).

## The Interface

- **Top Bar**: Contains main actions like **Templates**, **Save**, and **Export**.
- **Left Sidebar (Elements)**:
  - **Toolbar**: Drag and drop elements (Text, Button, Image) onto your page.
  - **DOM Tree**: See the structure of your email elements.
- **Center (Canvas)**: This is your email. Click anywhere to type or edit.
- **Right Sidebar (Properties)**: Change colors, fonts, and settings for the selected element.
