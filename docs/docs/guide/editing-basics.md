# Editing Basics

## Adding Content

- **Drag and Drop**: Grab an item from the **Elements Toolbar** (left side) and drop it into your email.
- **Text**: Double-click any text to start typing. You can use standard keyboard shortcuts like `Ctrl+B` for bold.

## Customizing Elements

Click on any element (like a button or image) to select it. The **Properties Panel** on the right will update.

### Text & Fonts

- **Typography Group**: Change font size, weight (boldness), alignment, and color.

### Colors & Backgrounds

- **Background Group**: Set a background color for sections or buttons.
- **Border**: Add rounded corners (`border-radius`) or outlines.

### Links

Select any link (or text that is a link):

1. Look for **Link Settings** in the Properties Panel.
2. Enter the **URL** (e.g., `https://google.com`).
3. Check **New Tab** if you want it to open in a new window.

### Images

Click an image to select it. In the Properties Panel:

- **Source (src)**: Paste the URL of your image.
- **Size**: Adjust width and height.
